# Глоссарий терминов

**Проект**: Joy Vision Calculator

---

## Бизнес-термины

| Термин | Описание |
|--------|----------|
| **Безрамное остекление** | Система остекления без видимых рам, стёкла крепятся на профилях |
| **Створка** | Отдельная стеклянная панель в системе |
| **Проём** | Открытое пространство, которое нужно остеклить |
| **КП** | Коммерческое предложение — документ с ценой для клиента |
| **Разблюдовка** | Документ с перечнем комплектующих для склада/производства |

---

## Системы остекления

| Система | Описание |
|---------|----------|
| **Slider L** | Параллельно-сдвижная система, створки двигаются вдоль проёма |
| **Slider X** | Усиленная версия Slider L для тяжёлых створок |
| **JV Line** | Система с парковкой, створки складываются в сторону |
| **JV Zig-Zag** | Гармошка, створки складываются зигзагом |

---

## Комплектующие

| Термин | Описание |
|--------|----------|
| **Профиль** | Алюминиевая направляющая (верхняя, нижняя, боковая, створочная) |
| **Заглушка** | Пластиковая деталь, закрывающая торцы профилей |
| **Роликовая каретка** | Механизм для движения створки по профилю |
| **Демпфер** | Амортизатор для мягкого закрывания |
| **Уплотнитель** | Резиновая/силиконовая лента для герметизации |
| **Ручка** | Элемент управления створкой (круглая, светопрозрачная, кноб) |
| **Задвижка/Защёлка** | Фиксатор створки в закрытом положении |

---

## Параметры расчёта

| Параметр | Описание |
|----------|----------|
| **Ширина проёма** | Горизонтальный размер в мм |
| **Высота проёма** | Вертикальный размер в мм |
| **Количество створок** | Число стеклянных панелей |
| **Направление открывания** | Влево, вправо, от центра |
| **Тип края** | Боковой профиль / без профиля / стык 90° |
| **RAL** | Цвет покраски по международной шкале RAL |
| **Толщина стекла** | Обычно 10 мм (закалённое) |

---

## Технические термины

| Термин | Описание |
|--------|----------|
| **3 дорожки / 5 дорожек** | Количество направляющих в верхнем профиле |
| **WAL mode** | Режим SQLite для параллельного доступа |
| **ORM** | Object-Relational Mapping (SQLAlchemy) |
| **Blueprint** | Модульная структура Flask приложения |
| **Webhook** | URL для API запросов к Битрикс24 |

---

## Статусы заказа

| Статус | Описание |
|--------|----------|
| **draft** | Черновик, заказ формируется |
| **confirmed** | Подтверждён клиентом |
| **in_production** | В производстве |
| **completed** | Выполнен |
| **cancelled** | Отменён |
